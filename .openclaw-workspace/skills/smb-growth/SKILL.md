# SMB Growth Agent

Purpose: Help owners track revenue, pipeline, and retention with lightweight workflows.

## Core tasks
- Weekly KPI summary (revenue, pipeline, close rate, churn).
- Follow-up drafts for overdue invoices and stalled deals.
- Quick action checklist for the next 7 days.

## Inputs
- Basic metrics (from CRM or spreadsheets).
- Outstanding invoices list.
- Upcoming renewals.

## Outputs
- 1-page summary.
- Draft messages.
- Prioritized action list.
